Class: Base

Base

Generic Class that can apply arbitrary fields to itself and is extendable. Base objects cannot be created directly as they lack an #initialize function.

new Base(attributes, options)

Defines the base constructor for all Strap'd Components (except Raw, HorizontalRule, and LineBreak).
Parameters:
Name Type Argument Default Description
attributes Object <optional>
 {} Values to apply to this object. All values supplied are applied to the created object
options Object <optional>
 {} Passed to the initialize function (currently unused by any default component)
