Class: Raw

Raw

A simple wrapper class providing render capability to an artibrary string of text. Raws function as Components with the notable exception of only retaining or rendering its body field.

new Raw(attrs)

Generates a renderable entity that acts as a simple passthrough for the attributes it's given.
