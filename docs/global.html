Typify(component, options)

Decorates the given component with the necessary variables and methods to handle being typed. A typed object is one that has a "base" and is then further modified with a "type". E.G. An alert can be an "error" message (and have a "type" of "error") This method does not overwrite any variables set on the decorated component unless it already has a setType property (which it shouldn't >:[)
Parameters:
Name Type Description
component [Component] the component to be decorated,
options [Object] the default values of type, base, and types can be defined in this object
